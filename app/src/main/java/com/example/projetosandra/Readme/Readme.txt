# Projeto de Reservas de Restaurante

Este projeto é um protótipo de um aplicativo Android desenvolvido em **Java** utilizando **Android Studio**.
O aplicativo permite que usuários façam reservas em um restaurante, visualizem suas reservas, editem ou cancelem conforme necessário.

## Funcionalidades

- **Cadastro e Login de Usuários**:
  - Criação de conta com nome, email, CPF e senha.
  - Autenticação de usuário via email e senha.

- **Gerenciamento de Reservas**:
  - Fazer novas reservas com data e número de pessoas.
  - Visualizar reservas feitas pelo usuário.
  - Editar informações da reserva.
  - Cancelar reservas existentes.
